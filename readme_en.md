# Auspicious Warrior - Tibetan-Chinese Intelligent Document Translator

This tool is a batch translation solution for Tibetan documents built on large language model APIs, designed for large-scale translation scenarios including Tibetan ancient books, academic literature and religious classics. It supports direct upload of local `.txt` and `.docx` files, allows free configuration of third-party LLM endpoints, and provides customizable parameters such as translation prompts and characters per batch. While preserving the original paragraph structure and typography, it automates Tibetan-to-Chinese translation for long documents, greatly reducing the cost of translating and organizing Tibetan materials.

## Key Features

- Native multi-format support: Import txt and docx documents directly, no manual splitting or format conversion required
- Fully customizable API configuration: Works with any OpenAI-compatible LLM endpoints, with customizable API URL, API key and model name
- Custom translation prompts: Configure scenario-specific prompts to define translation style, terminology standards and format retention rules
- Flexible batching strategy: Adjust characters per batch to fit the context window of different models, balancing translation efficiency and API cost
- Visual translation progress: Real-time display of batch progress and completion status, with side-by-side preview of original text and translation
- Layout preservation: Retains original paragraph structure in output, convenient for subsequent proofreading, annotation and secondary editing
