基于大模型 API 的藏文文档批量翻译工具，支持多格式文件、自定义接口与翻译规则。

吉祥勇士心・藏汉智能文档翻译工具
本工具是一款基于大语言模型 API 的藏文文档批量翻译解决方案，专为藏文古籍、学术文献、宗教典籍等内容的规模化翻译场景打造。工具支持本地 .txt、.docx 格式文件直接上传，可自由配置第三方大模型接口，支持自定义翻译提示词、分批字符数等参数，在保留原文段落结构与排版的前提下，自动化完成长文档的藏译汉处理，大幅降低藏文资料的翻译与整理成本。
核心特性
多格式原生支持：直接导入 txt、docx 文档，无需手动拆分与格式转换
全自定义 API 配置：支持填写任意兼容 OpenAI 格式的大模型接口，可自定义 API 地址、密钥与模型名称
翻译提示词自定义：可根据翻译场景配置专属提示词，指定译文风格、专业术语规范与格式保留要求
灵活分批策略：可自定义单批翻译字符数，适配不同模型的上下文窗口限制，平衡翻译效率与调用成本
翻译进度可视化：实时展示批次翻译进度与完成状态，支持原文与译文对照预览
排版结构保留：翻译过程还原原文段落结构，输出结果便于后续校对、标注与二次编辑
